# 数字货币价格查询与提醒插件使用文档
## 📌 插件概述
这是一个基于 AstrBot 框架开发的数字货币价格查询和提醒插件，支持实时查询 BTC、ETH 等主流数字货币价格，并支持设置价格突破提醒。

**插件信息**
* 名称：currency
* 描述：数字货币价格查询和提醒

## 🔧 安装配置
### 1. 安装插件
将插件文件放置到 AstrBot 的插件目录中，重启机器人即可自动加载。

### 2. 配置文件
配置文件路径：`data/config/astrbot_plugin_currency_config.json`

```json
{
  "check_interval": 1800
}
```

| 配置项 | 类型 | 默认值 | 说明 |
|-----|----|-----|-----|
| check_interval | int | 1800 | 价格检查间隔（秒） |

⚠️ **注意**：设置提醒前请先清空已有的提醒，否则可能生成多个循环任务，容易被封 IP。

## 📖 命令列表
### 1. 查询实时价格
**暂时支持一下货币**
* /BTC
* /ETH
* /DOGE

**返回示例：**

```text
💰 BTC/USDT 实时价格
📊 最新价格: $43,250.50
📈 24h涨跌: +2.35%
🔥 7d涨跌: +5.12%
🕐 更新时间: 2024-01-15 14:30:25
```

### 2. 价格提醒管理
**/提醒**

设置价格突破提醒。

**格式：**` /提醒 <价格> <币种> <上/下>`

|参数|说明|示例|
|-|-|-|
|价格|目标价格（数字）|50000|
|币种|数字货币名称（不区分大小写）|BTC
方向|"上" 或 "下"|上

**示例：**

```text
/提醒 50000 BTC 上
```
当 BTC 价格上涨到 $50,000 时触发提醒

```text
/提醒 40000 ETH 下
```
当 ETH 价格下跌到 $40,000 时触发提醒

**返回示例：**

```text
✅ 已设置提醒！
币种: BTC
当前价格: $43,250.50
目标价格: $50,000.00
提醒条件: 价格向上突破
检查间隔: 1800秒
```
> ⚠️ 合理性检查：系统会自动检查提醒条件是否合理
> * 设置"上"突破时，目标价格必须高于当前价格
> * 设置"下"突破时，目标价格必须低于当前价格

**/提醒列表**
查看当前用户设置的所有价格提醒。

**格式：** `/提醒列表`

**返回示例：**

```text
📋 您的提醒列表
1. BTC 向上突破 $50,000.00
2. ETH 向下突破 $40,000.00

取消提醒：/取消 <价格> <币种> <上/下>
```

**/取消**
取消指定的价格提醒。

**格式**： `/取消 <价格> <币种> <上/下>`

**示例：**

```text
/取消 50000 BTC 上
```
**返回示例：**

```text
✅ 已取消 BTC 向上突破 $50,000.00 的提醒
```
**智能提示：**
如果找不到完全匹配的提醒，系统会列出相似的提醒供您选择：

```text
❌ 未找到 BTC 向上突破 $51000.00 的提醒

您当前 BTC 向上突破的提醒有：
  • $50000.00
```

## 🔄 数据持久化
插件会自动保存用户设置的价格提醒，重启机器人后会自动恢复。

**数据文件路径**： `data/plugin_data/currency/reminders.json`

```json
[
  {
    "user_id": "user123",
    "symbol": "BTC",
    "price": 50000,
    "direction": "上",
    "umo": "unified_msg_origin"
  }
]
```

## ⚠️ 注意事项
1. **IP 限制**：频繁设置提醒可能产生多个检查循环，建议先清空旧提醒再设置新提醒，避免被封 IP

2. **检查间隔**：价格检查间隔默认为 1800 秒（30 分钟），可通过配置文件修改

3. **币种支持**：目前支持 BTC、ETH 等主流币种（以 CoinGlass API 支持为准）

4. **提醒触发**：触发后提醒任务会自动删除，不会重复提醒

## 🐛 常见问题
**Q: 设置提醒后没有收到通知？**

**A**: 可能原因：
* 价格尚未达到目标值
* 检查间隔未到（默认 30 分钟检查一次）
* 网络连接问题导致 API 请求失败

**Q: 如何修改检查频率？**

**A**: 编辑配置文件 `data/config/astrbot_plugin_currency_config.json`，修改 `check_interval` 值（单位：秒）

**Q: 支持哪些数字货币？**

**A**: 理论上支持 CoinGlass API 提供的所有币种，建议使用 BTC、ETH 等主流币种进行测试。

**Q: 重启机器人后提醒会丢失吗？**

**A**: 不会，插件会自动从 `reminders.json` 文件中恢复所有提醒任务。
